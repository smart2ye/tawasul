# دليل تشغيل ورفع خادم سحابة تواصل بلس على Render.com 🚀☁️

مرحباً بك! لقد قمنا بتجهيز خادم سحابي كامل لـ **تواصل بلس** مكتوب بلغة Node.js وإطار عمل Express. هذا الخادم يوفر الموقع الرسمي التعريفي، صفحة تحميل التطبيق APK، لوحة إدارة وتحكم مدمجة للمسؤولين، وجميع برمجيات الـ API الخاصة بحسابات المستخدمين والنسخ الاحتياطي المشفر وتذاكر الدعم والشكاوى وبث التنبيهات.

فيما يلي دليل خطوة بخطوة باللغة العربية لرفع هذا المشروع وتشغيله مجاناً بالكامل.

---

## 🛠️ المتطلبات الأساسية
1. حساب مجاني على موقع **GitHub**.
2. حساب مجاني على موقع **Render.com**.
3. استخدام تطبيق **Termux** على هاتفك للرفع والتعديل.

---

## 📝 الخطوة 1: تهيئة ورفع المشروع على GitHub باستخدام Termux

افتح تطبيق **Termux** ونفذ الأوامر التالية بالترتيب لتهيئة Git ورفع الكود بالكامل إلى مستودعك الخاص:

```bash
# 1. تثبيت حزمة git إن لم تكن مثبتة
pkg install git -y

# 2. تهيئة المستودع المحلي داخل مجلد المشروع
git init

# 3. إعداد هويتك الشخصية في Git (استبدل البيانات ببياناتك الحقيقية)
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

# 4. إضافة الملفات للمستودع وتجهيزها للرفع
git add .

# 5. عمل أول Commit للكود
git commit -m "إطلاق خادم تواصل بلس وموقع الخدمات السحابية"

# 6. ربط المستودع المحلي بمستودعك الجديد على GitHub
# (تأكد من إنشاء مستودع فارغ على GitHub أولاً باسم tawasul-plus-app)
git remote add origin https://github.com/USERNAME/tawasul-plus-app.git

# 7. رفع الكود للمستودع الرئيسي
git branch -M main
git push -u origin main
```

*ملاحظة: سيطلب منك GitHub إدخال اسم المستخدم وكلمة المرور (أو رمز الوصول الشخصي Personal Access Token) لإتمام عملية الرفع.*

## ☁️ الخطوة 2: إنشاء وتهيئة الخدمة على Render.com

موقع **Render** يدعم تشغيل خوادم Node.js مجاناً وبسهولة تامة. اتبع الخطوات التالية:

### أ) ربط مستودع GitHub بـ Render
1. سجل الدخول إلى لوحة التحكم في [Render Dashboard](https://dashboard.render.com).
2. اضغط على الزر الأزرق **New +** ثم اختر **Web Service**.
3. قم بربط حسابك بـ GitHub واشحن المستودع `tawasul-plus-app` الذي قمت برفعه للتو.

### ب) إعدادات خادم الويب (Web Service Settings)
أثناء تهيئة الخدمة، تأكد من ملء الحقول التالية بدقة:
* **Name**: `tawasul-plus` (أو أي اسم تفضله ليكون رابط موقعك).
* **Region**: اختر أقرب منطقة لك (مثال: `Frankfurt` أو `Singapore`).
* **Runtime**: `Node`.
* **Root Directory**: `server` (⚠️ مهم جداً! هذا يخبر Render أن الخادم يقع داخل مجلد `server` وليس في المجلد الرئيسي لتطبيق أندرويد).
* **Build Command**: `npm install`
* **Start Command**: `npm start`
* **Instance Type**: اختر الخطة المجانية **Free**.

### ج) إعداد متغيرات البيئة (Environment Variables)
اضغط على تبويب **Advanced** أو **Environment** وأضف المتغيرات التالية:
1. مفتاح `JWT_SECRET`: اكتب قيمة نصية عشوائية قوية جداً لتشفير جلسات مستخدميك.
2. (اختياري) في حال قمت بإنشاء قاعدة بيانات PostgreSQL مجانية على Render، قم بنسخ رابط الاتصال الداخلي ولصقه في مفتاح `DATABASE_URL`.
   * *ملاحظة ممتازة: في حال لم تقم بتوفير `DATABASE_URL`، سيقوم الخادم تلقائياً بإنشاء قاعدة بيانات SQLite محلية كملف داخل الخادم باسم `database.sqlite` وتعمل فوراً دون الحاجة لأي إعدادات إضافية!*

اضغط الآن على **Create Web Service** وانتظر دقيقتين ليقوم Render ببناء وتشغيل موقعك الإلكتروني بنجاح! 🎉

## 🌐 الخطوة 3: تصفح موقعك واستخدام لوحة الإدارة

بمجرد اكتمال البناء، سيمنحك Render رابطاً حياً لموقعك الإلكتروني (على سبيل المثال: `https://tawasul-plus.onrender.com`).

* **رابط الموقع والخدمات للمستخدمين**: `https://tawasul-plus.onrender.com`
* **رابط بوابة الإدارة والمسؤولين**: `https://tawasul-plus.onrender.com/admin`
* **بيانات دخول المسؤول الافتراضية**:
  * **البريد الإلكتروني**: `admin@tawasulplus.com`
  * **كلمة المرور الافتراضية**: `admin123456`

⚠️ *تنبيه أمني: بعد تسجيل الدخول لأول مرة في لوحة الإدارة، يرجى تغيير كلمة المرور أو إنشاء حساب مسؤول جديد لحماية خادمك من الاختراق.*

## 📱 الخطوة 4: ربط تطبيق أندرويد بالخادم الحقيقي الجديد

الآن بعد أن أصبح خادمك جاهزاً وموقعك حياً على شبكة الإنترنت، يمكنك ربط تطبيق الأندرويد ليتواصل مع خادمك السحابي الحقيقي بدلاً من الخادم الافتراضي الداخلي:

1. افتح تطبيق **Tawasul Plus** في بيئة العمل.
2. اذهب إلى ملفات الضبط (أو واجهة الإعدادات السحابية) وقم بتبديل رابط الـ API للرابط الخاص بك على Render: `https://tawasul-plus.onrender.com/api`.
3. قم ببناء نسخة الـ APK وتوزيعها على مستخدميك، واستمتع بتجربة اتصال ونسخ احتياطي حقيقية وسلسة!
